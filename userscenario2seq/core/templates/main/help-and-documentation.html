Tutorial
Tutorial is a page that explains the procedures or steps that can be done on the features that exist in this application. This is intended to make it easier for users to obtain information in the use of this website.
Create Project
Create Project is a page that serves to create a new project on this website. On the create project page there are several forms that the user must fill out, such as the project name and project description.
List Project
List Project is a page that displays all the projects that have been created in that account. On this page there is a search feature that you can use to find the project you need. In addition, on this page each project will display edit and delete features that can be used if you want to change the contents of the project or delete the project.
Detail Project
Detail Project is a page that displays the detailed contents of a project. This page contains the names of features that have scenarios created in a project. These details can be found in each project in the project list.
Add Feature
Add Feature is a feature that is used to add features that contain forms that must be filled out based on a user story scenario that will be generated into a sequence diagram. In filling out the form there are conditions that must be considered in writing so that the results produced are maximized.
Add Scenario
Add Scenario is a feature that is used to add alternative scenarios apart from the main scenario. In the alternative scenario there is only one condition that can be added, namely the “then” condition.
Generate Sequence
Generate Sequence is a feature that will be used to convert a previously created scenario into a sequence diagram. This generator can be found on the project detail page for each feature that has been created
