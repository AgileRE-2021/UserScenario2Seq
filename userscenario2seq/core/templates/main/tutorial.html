Halaman ini berisi tentang instruksi penggunaan aplikasi.

  • Tutorial

First, click the "Create Project"

First, click the "Create Project" button inside the sidebar.

Second, fill the project's information (see the steps below)
  1. Insert the project's name
  2. Insert the project's description
  3. Click the "Submit" button
Your project is created!

Your new project will be displayed in List Project page. You can use the search feature to find any specific project.

Previous Next

First, click the "Detail" button on a specific project

You can also delete the project by clicking the "Delete" button.

Second, click the "Add Feature" button

You can also edit your project name and description by clicking the "Edit Project" button.

Third, fill the feature's information (see the steps below)
  1. Insert the feature's name
  2. Insert the user story (pay attention to the format)
Fourth, complete the normal scenario (see the steps below)
  1. Insert the normal scenario name (ex: Login Success)
  2. Choose the type of each condition (you can choose on the additional condition)
  3. Fill the condition based on it's type (see the instruction above)
  4. You can add new condition (When/Then) by clicking the "Add Condition" button
  5. You can remove the additional condition by clicking the "Decrease Condition" button
Fifth, you can also add alternative scenarios (see the steps below)
  1. Click the "Add Scenario" button to add alternative scenario
  2. You can delete the alternative scenario by clicking the "Delete Scenario" button
  3. If you finish already, click the "Submit" button
You can see the new feature at the project's detail page

You can also search the specific feature

First, click the "Generate Sequence" button on the feature

You can also delete or edit the feature by clicking on the "Edit" or "Delete" button.

Then, a new tab with the sequence diagram in it will be opened

You can also download it by right click on the image then choose "Save Image As".
